# Customer Theme Style Guide

## Colors

| Element | Value |
|---------|-------|
| Primary | `#1e366c` |
| Background | `#f3f4f6` |
| Text Primary | `#333` |
| Text Secondary | `#6b7280` |
| Success | `#28a745` |
| Danger | `#dc3545` |

## Components

- **Header**: Primary background, white text/icons
- **Cards**: White, subtle shadow, 12px radius
- **Buttons**: 8px radius, 600 font-weight

## Guidelines

- Minimal design with white space
- Primary color for header and titles only
- Neutral grays for most UI elements
- Icons: Bootstrap Icons

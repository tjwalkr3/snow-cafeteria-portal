# Management Theme Style Guide

## Colors

| Element | Value |
|---------|-------|
| Primary | `#1e366c` |
| Accent | `#1b6ec2` |
| Text on Dark | `#fff` |
| Hover (dark) | `rgba(255,255,255,0.08)` |
| Active (dark) | `rgba(255,255,255,0.12)` |

## Components

- **Sidebar**: Primary background, white icons/text
- **Nav Items**: Centered icons (collapsed), icon + label (expanded)
- **User Card**: Avatar circle with accent color

## Guidelines

- Dark sidebar, light content area
- Accent blue for interactive elements
- Icons: Bootstrap Icons, 1.5rem size
- Smooth transitions (0.2s ease)
